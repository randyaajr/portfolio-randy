Boutique Ado An E-commerce Website Built With Django

Project information

  • Category: Full Stack
  • Project date: 11 February, 2020
  • Project URL: www.BoutiqueAdo.com

Boutique Ado is a full stack application. This E-commerce Website was built with Balsamiq, HTML, CSS, Bootstrap, FontAwesome, JavaScript, JQuery, Python (Jinja, Django, Django Crispy Forms, Psycopg2, Gunicorn), Stripe, SQlite3, PostgreSQL and AWS S3 Bucket.
