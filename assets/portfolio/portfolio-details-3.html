InfoFlix A Web Service To Obtain Movie Information

Project information

  • Category: Front-End, API
  • Project date: 20 November, 2020
  • Project URL: www.InfoFlix.com

InfoFlix is a Web App which sends API requests to a movie database. InfoFlix is built with HTML, CSS, BootStrap, JavaScript , jQuery and Axios. InfoFlix sends HTTP requests to OMDb API for movie information.
